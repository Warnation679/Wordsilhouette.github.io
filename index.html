Word Silhouette

By Francisco Figuereo

------------------------------------------

Word silhouette is a program that generates words to track body movement

It works using ML5 bodypose to help track each main body part

After generating a set amount of words the code will designate each main point a certain amount of words to follow it

These will will follow the body in real-time making a silhouette of the person

It's promgramed to recognize the full body so for the full experience it's best to capture the full body

Sources:

https://docs.ml5js.org/#/reference/bodypose
